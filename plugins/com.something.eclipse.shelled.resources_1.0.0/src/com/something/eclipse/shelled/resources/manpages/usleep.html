SYNOPSIS

       usleep [ usec ]

DESCRIPTION

       usleep  pauses  for  the  number of usec microseconds. The
       default is 1 microsecond.  If 0 microseconds are specified
       sched_yield(2) is called.

BUGS

       The  usleep program uses the usleep(3) function and there­
       fore shows the same weaknesses by any system activity.

SEE ALSO

       usleep(3), sleep(1), sleep(3), sched_yield(2).
