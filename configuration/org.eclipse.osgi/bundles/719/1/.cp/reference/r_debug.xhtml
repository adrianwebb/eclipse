Quick Start Guide

  1. Launch the R Console in Debug mode ()
  2. Set breakpoints in your functions
  3. Submit your functions by using a command supporting the debugger
  4. Invoce the code
  5. If R hit a breakpoint, the Debug Perspective will be opened with
    • the Debug View showing the call stack (help by JDT),
    • the Variables View showing the variables in the selected stack frame (help by JDT),
    • the Breakpoints View (help by JDT),
    • the known Console View
    • and more...,
    and you can step through your code using the execution control commands.
