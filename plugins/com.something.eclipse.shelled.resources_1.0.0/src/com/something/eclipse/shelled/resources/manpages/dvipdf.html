SYNOPSIS

       dvipdf [ options ] input.dvi [ output.pdf ] ...

DESCRIPTION

       This script invokes dvips(1) with the -q option, and pipes
       its output into gs(1) with the following options:

                 -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite

       as  well as -sOutputFile and any options from the command-
       line.

SEE ALSO

       gs(1), dvips(1)
