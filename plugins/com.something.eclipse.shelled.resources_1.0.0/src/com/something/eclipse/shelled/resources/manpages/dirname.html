SYNOPSIS

       dirname NAME
       dirname OPTION

DESCRIPTION

       Print  NAME  with its trailing /component removed; if NAME
       contains no /'s, output `.' (meaning  the  current  direc­
       tory).

       --help display this help and exit

       --version
              output version information and exit
