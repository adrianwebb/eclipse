Description

       The  mdel  command  is  used to delete an MS-DOS file. Its
       syntax is:

       mdel [-v] msdosfile [ msdosfiles ...  ]


       Mdel deletes files on an MS-DOS filesystem.

       Mdel asks for verification prior to removing  a  read-only
       file.
