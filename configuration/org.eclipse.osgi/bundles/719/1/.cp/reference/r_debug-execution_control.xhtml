Execution Control for the R Debugger

Commands

Note that most listed commands depends on the selection like the debug target (R engine), thread or stack frame of the Debug view in the currently active workbench window.

Name Description Availability
Resume

Resumes the execution of the currently suspended selected debug target or thread.

 Main menu (Run)
View/Main toolbar (Debug)1)
Global command
F8
Suspend

Suspends the execution of the currently selected debug target or thread.

Note: this command does not work in all situations and R versions.

 Main menu (Run)
View/Main toolbar (Debug)1)
Global command
Step Into

Note: this command does not work for R, use Step Into Selection instead.

 Main menu (Run)
View/Main toolbar (Debug)1)
Global command
F5
Step Into Selection

Steps into the function call which is currently selected in the R editor.

The R editor must be active. The selected function is marked to debug and the execution resumes until the method is invoked.

The same action can be triggered using the editor hyperlink action "Step Into Selection" with the default modifier Ctrl+Alt.

 Main menu (Run)
Global command
Ctrl+F5
Step Over

Steps over the current statement.

The execution resumes with the current statement in the selected stack frame and will suspend at the next statement in the same function.

 Main menu (Run)
View/Main toolbar (Debug)1)
Global command
F6
Step Return

Steps out of the current function.

The execution resumes the current function of the selected stack frame and will suspend after exiting the function in the stack frame below.

 Main menu (Run)
View/Main toolbar (Debug)1)
Global command
F7
Terminate

Terminates the selected element (debug target/R engine, process, ...).

If the R thread is suspend, the terminate action for the R engine/R thread exits the current debugging state and execution, but does not quit R. Trigger the terminate command repeatedly to finally quit R.

Note: For some elements types the command simply kills the R process. It is more safe to use the Terminate command in the Console View!

 Main menu (Run)
View/Main toolbar (Debug)1)
Context menu (Debug View)
Global command
Ctrl+F2
Disconnect

Disconnects the console from the remote R engine like the disconnect command in the Console View.

 Main menu (Run)
Context menu (Debug View)
Global command
Terminate and Relaunch

Terminates the selected debug target and relaunches it.

Note: This command simply kills the local R process and is not recommend for local R debug targets!

 Context menu (Debug View)
Global command
Terminate and Remove

Terminates the selected debug target and removes it from list of launches.

Note: This command simply kills the local R process and is not recommend for local R debug targets!

 Context menu (Debug View)
Terminate/Disconnect All

Terminates all active launches.

Note: This command simply kills the local R process and is not recommend for local R debug targets!

 Context menu (Debug View)
